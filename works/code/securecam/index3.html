Ip камеры видеонаблюдения уличные

Ip камеры видеонаблюдения уличные – эффективное решения для организации видеонаблюдения вне помещений, а также в помещениях со сложными условиями эксплуатации (перепады температур, пыль, плохая освещенность). Основное отличие таких камер видеонаблюдения - полусферический корпус, позволяющий регулировать направление обзора в широких пределах. Защитный колпак может быть выполнен из дымчатого стекла (в т.ч. вандалостойкого) скрывающего реальное направление обзора. Основное применение ip камер видеонаблюдения уличных – неотапливаемые склады, охрана периметра, парковки, производственная территория.

Внимание! Для товаров, помеченных как спец.цена необходимо согласовывать наличие с менеджером!
